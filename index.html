Projects

Swoop Marketplace

Swoop Marketplace was a 3 tier cloud native web application that I helped create along with 3 other people. This was a semester long project which was done as part of the course work for ITMT-430. We used Github as our version control and deployed our application to a proxmox server hosted on IIT's Rice Campus. We used the laravel framework to develop a mock marketplace that users could sign in to and view items that were for sale. Users who were administrators could also upload images to create their own listings.

Our site featured support for Google Authentication. Users were able to login with their Hawk accounts through google. After users logged in they could browse the catalog of products that were available on the site and add them to the shopping cart to simulate browsing and checking out on a real store page. Our catalog page also featured a search bar which users could type the name of a product. This would create a SQL Query which looked for products in our database and update the site with any result that matched.
